Kardec
Design System

Sistema de design editorial para o Projeto Allan Kardec. Tokens, componentes CSS e Blade Components para Laravel, construídos sobre uma estética de arquivo histórico refinada.

v 1.0 Laravel Blade CSS Custom Properties Alpine.js ready

EstruturaArquivos CSS

Importar em ordem
/* resources/css/app.css — ou via Vite */ @import 'kardec-tokens.css'; /* 1. Custom properties */ @import 'kardec-base.css'; /* 2. Reset + tipografia + utilitários */ @import 'kardec-components.css'; /* 3. Todos os componentes */
ArquivoResponsabilidadeAlterar aqui para...
kardec-tokens.cssCSS custom properties (valores brutos e semânticos)Rebranding completo — mudar paleta, fontes, raios
kardec-base.cssReset, tipografia global, utilitários, animaçõesComportamento de scroll, estilos de foco, utilitários
kardec-components.cssTodos os componentes UI com classes .k-*Aparência de um componente específico

Laravel BladeComo usar

Instalação
# Copiar arquivos CSS para resources/css/ cp kardec-tokens.css resources/css/ cp kardec-base.css resources/css/ cp kardec-components.css resources/css/ # Copiar componentes para resources/views/components/ cp blade/components/*.blade.php resources/views/components/ cp blade/layouts/app.blade.php resources/views/layouts/ # Registrar CSS no vite.config.js input: [ 'resources/css/kardec-tokens.css', 'resources/css/kardec-base.css', 'resources/css/kardec-components.css', 'resources/js/app.js', ]
Exemplo de página completa
<x-layouts.app title="Início"> <x-hero eyebrow="Projeto Allan Kardec · UFJF" title="Manuscritos <em>&amp; Transcrições</em>" subtitle="Acervo digital de documentos originais de Allan Kardec" scroll-to="manuscritos" > <x-slot:search> <x-search-box :action="route('pesquisar')" /> </x-slot:search> <x-slot:filters> <x-chip label="Todos" :active="true" /> <x-chip label="Cartas" value="carta" /> </x-slot:filters> <x-slot:stats> <x-stat number="600+" label="Manuscritos" /> <x-stat number="3" label="Acervos" /> </x-slot:stats> </x-hero> <x-section id="manuscritos" variant="light"> <x-section-header tag="Acervo" title="Manuscritos Recentes" link-text="Ver todos" :link-href="route('manuscritos.index')" /> <div class="k-ms-grid"> @foreach($manuscripts as $ms) <x-manuscript-card :id="$ms->identifier" :title="$ms->title" :acervo="$ms->acervo->name" :href="route('manuscritos.show', $ms)" /> @endforeach </div> </x-section> </x-layouts.app>
